# QuayPanel Default Theme

This is the built-in QuayPanel look and feel. Core React layouts and components are the fallback views when this theme is active.

## Forking

1. Copy `themes/default` to `themes/your-theme-id`
2. Update `theme.json` (`id`, `name`, tokens)
3. Optionally add `assets/theme.css` and/or a compiled `dist/index.js` that registers view overrides
4. Activate the theme in Admin → Themes
